# SP2 Resit Auction
![Auction App Screenshot](./assets/auction-screenshot.png)

A modern auction web application built for the **Noroff Frontend Development Semester Project 2 Resit**.  
Users can browse listings, create auctions, and place bids on items.

---

# Description

SP2 Resit Auction is a full-featured auction platform developed as part of the Noroff Frontend Development course.

The application allows users to browse available auction listings, view detailed item pages, and place bids. Registered users can create their own listings and manage their profile.

The project integrates with the **Noroff Auction API** for authentication, listings, and bidding functionality.

---

# Key Features

- Browse auction listings
- Search listings
- View detailed listing pages
- Create new listings
- Place bids on listings
- Register and login with a student email
- View user profile
- Update avatar
- View bids on listings
- Fully responsive design

# Getting Started

## Installing

Clone the repository:

```
git clone https://github.com/OH2021/SP2-Resit.git
```

Navigate into the project folder:

```
cd SP2-Resit
```

Install the dependencies:

```
npm install
```

---

## Running

Run the development environment:

```
npm run dev
```

The application will run locally depending on your configuration.
